## 一、项目概述

### 1.1 项目定位

**鑫琳医生** 是一款基于 AI 技术的智能医疗健康管理平台，通过 AI 医生分身为患者提供免费的初步问诊和医嘱执行监督服务，降低医疗门槛，提高患者医依从性。

### 1.2 愿景与使命

| | |
|---|---|
| **愿景** | 让每个家庭都有自己的 AI 健康管家 |
| **使命** | 通过 AI 技术降低医疗成本，提高健康管理效率 |

### 1.3 核心价值

- **对患者**：免费获得 7×24 小时 AI 问诊和医嘱监督
- **对医生**：通过 AI 分身处理重复性问诊，专注复杂病例
- **对保险公司**：降低赔付风险，数据支持精准定价
- **对药企/器械商**：精准触达目标患者群体

## 二、产品与服务

### 2.1 核心功能

#### (1) AI 免费问诊

- **多科室覆盖**：皮肤科、心血管科、骨科、内分泌科、消化科、神经科等
- **智能识别**：支持拍照上传症状，AI 图像识别辅助诊断
- **语音交互**：语音输入，AI 自动转文字并回复
- **病历生成**：问诊自动生成结构化病历，支持导出

#### (2) 医嘱执行监督 ⭐ 核心功能

- **医嘱创建**：医生/AI 开具个性化医嘱计划
- **智能提醒**：按时推送服药、检查、复诊提醒
- **打卡反馈**：患者拍照/语音反馈执行情况
- **执行率分析**：生成执行报告，医生可远程追踪

> **商业价值**：这是保险合作的数据基础

#### (3) VIP 远程问诊

- **真人医生**：付费接入真实专家医生
- **视频/语音**：支持远程视频问诊
- **预约排期**：灵活预约时间
- **优先响应**：VIP 用户优先接入

#### (4) 远程查房

- **医生发起**：医生发起查房任务
- **患者接收**：患者端收到通知，可响应
- **实时互动**：视频+语音实时沟通

#### (5) 健康档案

- **电子病历**：所有问诊记录归档
- **健康趋势**：症状、指标变化曲线
- **数据导出**：支持 PDF 导出，便于线下就医

### 2.2 技术优势

| 技术能力 | 说明 | 竞争优势 |
|----------|------|----------|
| 多智能体架构 | 每个科室独立 AI Agent | 专科化、更精准 |
| 视觉识别 | 皮肤病变、X 光片识别 | 辅助诊断能力 |
| 语音交互 | 本地 VAD + 云端 TTS | 老年人友好 |
| 医嘱监督 | 自动提醒+打卡追踪 | 提高依从性 |

## 三、商业模式

### 3.1 基本原则

```
患者端 ──────────────> 完全免费（获取用户量）
                         │
医生端 ──────────────> 付费招募（保证服务质量）
                         │
       ┌────────────────┼────────────────┐
       ▼                ▼                ▼
   VIP 问诊          广告收入          保险佣金
```

### 3.2 三大收入来源

#### 收入一：VIP 远程问诊费

| 用户类型 | 服务 | 收费模式 |
|----------|------|----------|
| 普通用户 | AI 免费问诊 | 免费 |
| VIP 用户 | 真人医生远程问诊 | 按次收费 或 包月订阅 |

**定价建议**：
- 按次：¥29 - ¥99 / 次（根据医生级别）
- 包月：¥199 / 月（无限次 AI + 2次真人问诊）

#### 收入二：广告收入

- **药品广告**：根据患者症状推荐相关药品，按点击/转化收费
- **医疗器械**：血压计、血糖仪等推荐，CPS 模式
- **体检服务**：与体检机构合作，按预约分成

**广告原则**：
- 明确标注"广告"标识
- 仅在相关场景推荐
- 不影响核心问诊体验

#### 收入三：保险佣金 ⭐ 创新模式

**合作模式**：

```
┌─────────────────────────────────────────────────────────────┐
│                     保险合作模式                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  患者 ──► 在鑫琳医生 ──► 医生开具医嘱 ──► 系统监督执行       │
│          购买保险       │                                    │
│                       ▼                                    │
│              医嘱执行率数据                                  │
│                       │                                    │
│                       ▼                                    │
│  保险公司 ◄─────── 鑫琳医生平台 ────────► 佣金收入          │
│    根据执行率          提供数据                             │
│    动态定价                                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**核心逻辑**：
- 按医嘱执行的患者：出险概率低，保险公司赔付少 → **盈利**
- 不按医嘱的患者：保险公司有权拒赔 → **降低损失**
- 鑫琳医生提供执行数据证明 → **收取数据服务费/佣金**

**收费模式**：
- 按保单金额的 3-5% 收取佣金
- 或按患者人数收取年费（¥50 - ¥100 / 人 / 年）

### 3.3 收入结构预测

| 收入来源 | 占比 | 说明 |
|----------|------|------|
| VIP 问诊 | 30% | 高毛利，直接收入 |
| 广告收入 | 20% | 流量变现，长尾收入 |
| 保险佣金 | 50% | 规模化后最大收入来源 |

## 四、市场分析

### 4.1 市场规模

#### 中国医疗健康市场

| 细分市场 | 规模（2024） | 增速 |
|----------|-------------|------|
| 在线问诊 | ¥500 亿 | 30%+ |
| 慢病管理 | ¥2000 亿 | 20%+ |
| 商业健康险 | ¥8000 亿 | 25%+ |

#### 目标用户

- **患者端**：全国慢性病患者（3亿+）、轻症患者
- **医院端**：二级及以下医院、基层医疗机构
- **保险端**：商业健康险公司

### 4.2 用户痛点

| 痛点 | 现状 | 鑫琳医生解决方案 |
|------|------|------------------|
| 排队时间长 | 平均 2-3 小时 | AI 秒级响应 |
| 医生资源不足 | 医患比 1:500 | AI 分身补充 |
| 医嘱难坚持 | 慢病依从性 < 50% | 智能监督+打卡 |
| 医疗信息混乱 | 百度看病癌症起步 | 专业 AI 引导 |
| 保险核保难 | 缺乏健康数据 | 医嘱数据支持 |

### 4.3 市场机会

1. **政策支持**：《"健康中国 2030"规划纲要》鼓励互联网+医疗
2. **老龄化加速**：60岁以上人口 2.9 亿，健康管理需求爆发
3. **慢病井喷**：糖尿病、高血压患者超 4 亿，长期管理需求强
4. **保险觉醒**：商业健康险增速快，需要风控数据

## 六、运营策略

### 6.1 冷启动策略

#### 阶段一：种子用户（0 - 10,000 用户）

- **目标人群**：慢性病患者（糖尿病、高血压）
- **获客渠道**：
  - 线下：社区义诊合作、医院地推
  - 线上：健康类公众号、小红书种草
  - 口碑：老用户邀请奖励

#### 阶段二：快速增长（10,000 - 100,000 用户）

- **内容营销**：健康科普短视频（抖音/视频号）
- **KOL 合作**：医生博主背书
- **保险合作**：与保险公司战略合作，导流用户

#### 阶段三：规模化（100,000+ 用户）

- **医院合作**：与基层医院系统对接
- **药企合作**：患者教育项目
- **广告投放**：精准投放获客

### 6.2 用户增长模型

```
新用户获取
    │
    ├─► AI 免费问诊（首因）
    │
    ├─► 医嘱监督计划（留存钩子）
    │       │
    │       └─► 每日打卡（形成习惯）
    │
    ├─► 付费转化
    │       ├─► VIP 问诊（即时需求）
    │       └─► 保险推荐（长期价值）
    │
    └─► 口碑分享（老带新）
```

### 6.3 关键指标 (KPIs)

| 指标 | 目标 | 说明 |
|------|------|------|
| DAU/MAU | > 30% | 日活/月活比，衡量粘性 |

## 九、风险与应对

### 9.1 主要风险

| 风险类型 | 风险描述 | 应对策略 |
|----------|----------|----------|
| **政策风险** | 互联网医疗监管收紧 | 持证经营，合规运营，只做辅助诊断 |
| **技术风险** | AI 误诊导致纠纷 | 明确免责条款，建议线下确诊 |
| **竞争风险** | 大厂进入 | 保险数据壁垒，快速扩张 |
| **数据风险** | 用户隐私泄露 | 严格数据加密，合规存储 |
| **保险合作** | 保险公司接受度低 | 先小范围试点，积累数据证明价值 |

### 9.2 合规要点

- 不涉及确诊开方，只做健康咨询和辅助
- 不存储敏感医疗数据，只存健康行为数据
- 明确 AI 仅供参考，不替代专业医疗
- 广告严格审核，不夸大疗效
